# 0x01. C - Variables, if, else, while

## Description

This project introduces basic C programming concepts including:
- Variables
- Arithmetic operators
- Conditional statements (if, else if, else)
- While loops

## Task 0: 0-positive_or_negative.c

This program assigns a random number to the variable `n` each time it runs and prints whether the number is:
- Positive
- Zero
- Negative

## Compilation

All programs are compiled using:

gcc -Wall -Werror -Wextra -pedantic -std=gnu89 file.c -o output_file

## Requirements

- Ubuntu 20.04 LTS
- No global variables
- Maximum 5 functions per file
- Betty style compliant
- All files end with a new line
