# 0x0B. C - Preprocessor

## Description
This project focuses on understanding the C preprocessor, macros, and header files. It introduces how to use object-like macros, function-like macros, and predefined macros in C programming.

---

## Learning Objectives
At the end of this project, you should be able to explain:

- What macros are and how they work
- How to use `#define` to create constants
- How to create function-like macros
- Common predefined macros in C (`__FILE__`, etc.)
- How header guards prevent multiple inclusions
