# 0x02. C - Functions, nested loops

##  Description

This project introduces the use of functions and nested loops in C programming.
It focuses on writing reusable code, understanding function prototypes, and working with control structures like loops.

---

##  Requirements

* Allowed editors: `vi`, `vim`, `emacs`
* Compilation:

  ```
  gcc -Wall -Werror -Wextra -pedantic -std=gnu89
  ```
* All files must end with a new line
* Code must follow **Betty style**
* No use of `printf`, `puts`, etc. (except where explicitly allowed)
* Use only `_putchar` for output
* No global variables
* Prototypes must be declared in `main.h`
* Do not push `_putchar.c`

---

## Files

| File                     | Description                               |
| ------------------------ | ----------------------------------------- |
| `0-putchar.c`            | Prints "_putchar" followed by a new line  |
| `1-alphabet.c`           | Prints the alphabet in lowercase          |
| `2-print_alphabet_x10.c` | Prints the alphabet 10 times              |
| `3-islower.c`            | Checks for lowercase character            |
| `4-isalpha.c`            | Checks for alphabetic character           |
| `5-sign.c`               | Prints the sign of a number               |
| `6-abs.c`                | Computes the absolute value of an integer |
| `7-print_last_digit.c`   | Prints the last digit of a number         |
| `8-24_hours.c`           | Prints every minute of the day            |
| `9-times_table.c`        | Prints the 9 times table                  |
| `10-add.c`               | Adds two integers                         |
| `11-print_to_98.c`       | Prints numbers from n to 98               |

---

##  Compilation Example

```
gcc -Wall -Werror -Wextra -pedantic -std=gnu89 _putchar.c file.c -o output
```

---

##  Usage

Run the compiled program:

```
./output
```
