# 0x05. Pointers, Arrays and Strings

## Description

This project focuses on understanding **pointers, arrays, and strings** in C. It contains small programs and functions that manipulate data using pointers, arrays, and strings without using standard library functions for output, except `_putchar`.

---

## Project Requirements

- Allowed editors: `vi`, `vim`, `emacs`
- Compiler: `gcc` using flags: `-Wall -Werror -Wextra -pedantic -std=gnu89`
- All files must end with a new line
- Functions must follow **Betty coding style** and documentation
- No global variables
- Maximum of 5 functions per file
- No use of `printf` or `puts` (except for testing with main)
- Only `_putchar` is allowed for output
- All prototypes must be in `main.h`
- A `README.md` is required in the root of the repository

---

## Tasks

- **0-reset_to_98.c**: Updates the value of an integer to `98` using a pointer.
- **1-swap.c**: Swaps the values of two integers using pointers.
- **2-strlen.c**: Returns the length of a string.
- **3-puts.c**: Prints a string using `_putchar`.
- **4-print_rev.c**: Prints a string in reverse.
- **5-rev_string.c**: Reverses a string.
- **6-puts2.c**: Prints every other character of a string.
- **7-puts_half.c**: Prints half of a string.
- **8-print_array.c**: Prints `n` elements of an array of integers.
- **9-strcpy.c**: Copies a string to another buffer.
- **10-atoi.c**: Converts a string to an integer.
